# 迁移设计

本项目从旧语雀社团助手的已验证底层思路中重写了语雀请求、TOC 路径投影、Markdown/frontmatter、SQLite 元数据和混合检索。没有复制旧项目代码；参考项目采用 AGPL-3.0-or-later，本项目也保留同一许可证。

未迁移成员绑定、画像、偏好、伙伴、学习路径、知识卡、学习缺口、订阅/推送、周报、长期记忆、协作网络、Token 限额、Git 自动操作、群聊旁听及所有旧品牌。

Webhook 经评估不纳入 v0.1：参考实现将 HTTP 服务、IP 白名单和大量同步/业务副作用耦合；本项目优先提供带锁的管理员手动全量同步。后续如实现，必须使用签名鉴权、幂等事件、受控队列和可停止后台任务。
